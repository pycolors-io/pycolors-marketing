## How it works

The billing system is built around Stripe as the **source of truth**.

Flow:

1. User clicks “Upgrade”
2. Stripe Checkout session is created
3. User completes payment
4. Stripe sends webhook events
5. Your database is updated (subscription, invoices)
6. UI reflects real billing state
