## Core architecture

The backend is structured around three layers.

### Data layer

- Prisma schema
- typed models
- migration-ready structure

### Application layer

- API and server-side patterns
- domain-oriented structure
- reusable logic boundaries

### Integration layer

- auth context
- billing context
- product access logic
- feature gating

That matters because a SaaS is not only screens.

It is rules, data, ownership, and access.
