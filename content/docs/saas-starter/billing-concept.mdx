Billing is a core SaaS workflow — but also one of the most complex to
wire.

Starter Free gives you the **complete billing UX surface** while keeping
implementation intentionally mocked.

------------------------------------------------------------------------

## Billing philosophy

Starter Free ships the **billing surface**, not the billing engine.

Included:

-   Plan display
-   Renewal info
-   Seats concept
-   Invoice history UI
-   Billing portal entrypoints

Not included:

-   Stripe integration
-   Webhooks
-   Subscription logic

------------------------------------------------------------------------

## Stripe integration flow

Backend:

``` bash
POST /api/billing/portal
```

Pseudo code:

``` ts
const session = await stripe.billingPortal.sessions.create({
  customer: stripeCustomerId,
  return_url: "https://yourapp.com/billing"
})
```

Frontend redirect:

``` ts
window.location.href = session.url
```

1.  User upgrades plan

4.  App loads updated plan

## Common pitfalls
